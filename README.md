# Clone-Pong

## ℹ️ Description

A clone of the classic Pong game, developed using Unity. The project contains a single-player and a two-players mode. There is also a button to mute the game and a resolution menu.

Link: [Play Game](https://zejsneto.itch.io/pong)<br>
